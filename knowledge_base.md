# База знаний бота портфолио

Этот документ описывает правила работы бота, требования к ZIP-архивам и возможные причины отказа.
Используйте его как справочник для ответов студентам.

## Назначение
Бот принимает ZIP-архивы портфолио студентов, проверяет структуру папок и имена PDF-файлов.
Если всё корректно, архив принимается и сохраняется. При ошибках возвращается понятный отчёт.

## Поддерживаемые архивы
- Только ZIP.
- RAR и другие форматы не поддерживаются.

## Структура архива (строго)
```
<GROUP>/<STUDENT_SHORT>/<DISCIPLINE_FULL>/<WORKTYPE_FOLDER>/<PDF_FILE>.pdf
```
Где:
- `<GROUP>` — папка группы (например, `ИТ-21б`).
- `<STUDENT_SHORT>` — строго формат `Фамилия ИО` (один пробел, далее 2 заглавные буквы).
- `<DISCIPLINE_FULL>` — ровно одна дисциплина из справочника.
- `<WORKTYPE_FOLDER>` — папка типа работ из белого списка.
- `<PDF_FILE>` — строго PDF-файл (никаких DOCX, ZIP внутри и т.п.).

Любые лишние файлы/папки (включая `__MACOSX`, `Thumbs.db`) приводят к отказу.

## Правила имени PDF
Шаблон: `Группа_ФИО_ДисциплинаКратко_ТипРаботыНомер.pdf`

Правила:
- Разделитель — строго `_` (4 блока).
- ФИО в имени файла должно совпадать с папкой `<STUDENT_SHORT>`.
- Группа в имени файла должна совпадать с корневой папкой.
- Сокращение дисциплины берётся из справочника (full -> short).
- Тип работы должен соответствовать папке типа работ.

Примеры типов работ:
- Лабораторные работы → `ЛР1`, `ЛР2`, ... (номер обязателен).
- Практические занятия → `ПЗ1`, `ПЗ2`, ... (номер обязателен).
- Практические работы → папка допустима, но в имени файла всё равно `ПЗN`.
- Курсовая работа → `КР` (номер запрещён).
- Расчётно-графическая работа → `РГР1`, `РГР2`, ... (номер обычно обязателен).

## Проверка PDF
- PDF должен открываться.
- Должно быть минимум 2 страницы.

## Результаты проверки
- **Принято** — структура корректна, имена корректны, PDF валидны.
- **Отклонено** — есть нарушения в структуре, именах или PDF.
- **Ручная проверка** — используется, если возникла техническая ошибка при отметке в журнале.

## Частые причины отказа
- Архив не ZIP.
- Неверное имя папки группы или студента.
- В архиве больше одной дисциплины.
- Лишние файлы/папки или неверное расширение.
- Неверный шаблон имени PDF.
- PDF не открывается.

## Что делать, если отказали
Скопируйте текст отчёта, исправьте указанные ошибки и отправьте архив снова.
